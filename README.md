# Parser for Navy #

It parses.

## Install ##

1. Ruby version 1.9+ (`rvm install 1.9.3`; RVM should do the rest)
2. Bundler: `gem install bundler`
3. `cd **parser-directory/**`
4. Use the ruby version recommended by RVM.
4. `bundle install`
5. If necessary, `(sudo) chmod +x parser.rb`
5. `./parser.rb --help`
